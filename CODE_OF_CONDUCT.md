# Contributor Covenant Code of Conduct

This project uses the [Contributor Covenant](https://www.contributor-covenant.org/)
Code of Conduct, version 2.1, with the additional expectation that contributors
respect the non-carceral, privacy-preserving norms described in this repo.

The full text of the Contributor Covenant v2.1 is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

For attribution, see:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/
